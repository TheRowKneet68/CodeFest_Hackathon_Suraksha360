ESP8266 NODEMCU + SIM800L + OLED + EMERGENCY BUTTON WIRING

IMPORTANT
- Connect every GND pin together: NodeMCU GND, SIM800L GND, OLED GND,
  button GND, and the negative terminal of the SIM800L power supply.
- Power the SIM800L from a separate stable 3.7V to 4.2V supply capable of
  handling approximately 2A current spikes. Do not use the NodeMCU 3.3V pin
  to power the SIM800L.

1. OLED DISPLAY (I2C, 128x64 SSD1306)
- NodeMCU D1 (GPIO5)  -> OLED SCL
- NodeMCU D2 (GPIO4)  -> OLED SDA
- NodeMCU 3V3         -> OLED VCC
- NodeMCU GND         -> OLED GND

2. EMERGENCY BUTTON
- NodeMCU D0 (GPIO16) -> One button terminal
- NodeMCU GND         -> Other button terminal
- The sketch uses INPUT_PULLUP, so no external pull-up resistor is needed.

3. NEO-M8N GPS SERIAL CONNECTION
- NEO-M8N TX          -> NodeMCU D5 (GPIO14)
- NodeMCU D6 (GPIO12) -> NEO-M8N RX through a voltage divider, if the GPS RX
  pin is used. GPS position reading only needs the NEO-M8N TX connection.
- NEO-M8N GND         -> Common GND

4. SIM800L SERIAL CONNECTION
- SIM800L TX          -> NodeMCU D7 (GPIO13) directly
- NodeMCU D8 (GPIO15) -> 1k ohm resistor -> SIM800L RX
- At the SIM800L RX point, connect a 2k ohm resistor from SIM800L RX to GND.

  NodeMCU D8 / GPIO15 (3.3V)
          |
         1k ohm
          |
          +----------> SIM800L RX
          |
         2k ohm
          |
         GND

5. SIM800L POWER
- SIM800L VCC         -> Positive terminal of separate 3.7V to 4.2V supply
- SIM800L GND         -> Negative terminal of that supply and common GND
- SIM800L TX/RX must share the same common GND with the NodeMCU.

6. UPLOAD NOTE
- The sketch uses Serial.swap(), so USB uploading remains on the normal
  NodeMCU USB serial pins while the SIM800L uses D7 and D8.
