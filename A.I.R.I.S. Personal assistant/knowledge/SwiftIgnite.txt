Full Name:

Project Type:
IoT-Based Vehicle Security and Smart Ignition System

Developer:
Ronit Baniya Gupta

Project Description:
Swift Ignite is a smart keyless bike ignition system designed to replace traditional mechanical ignition methods with a secure electronic control system. The system enables authorized users to unlock, lock, and start a motorcycle using wireless communication and embedded control hardware.

The project focuses on improving vehicle security, user convenience, and anti-theft protection while maintaining a cost-effective and practical design suitable for real-world deployment.

System Operation:

Unlock Command:
Command: I

Function:

* Unlocks ignition system.
* Enables bike starting functionality.
* Activates primary relay.

Start Command:
Command: R

Function:

* Starts motorcycle engine.
* Works only after successful unlock.
* Activates starter relay briefly.

Lock Command:
Command: S

Function:

* Disables ignition system.
* Prevents unauthorized engine starting.
* Returns system to secure state.
